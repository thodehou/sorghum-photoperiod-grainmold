# sorghum-photoperiod-grainmold

Reproducible code, data, and outputs supporting the paper:
**Stable genomic variants for photoperiodic flowering plasticity to enhance grain mold escape in sorghum**

### Repository structure
- `scripts/` – Main analysis scripts
- `functions/` – Custom R functions
- `data/` – Datasets used
- `outputs/` – Figures and generated outputs
